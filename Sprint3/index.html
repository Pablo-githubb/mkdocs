    INSTAL·LACIÓ DOMINI LDAP I UNIR CLIENT AL DOMINI

    1. INSTAL·LACIÓ DOMINI LDAP

    per a fer la base del domini (arbre de directori), a traves dels arxius .ldif o a traves de comanda.

    Fem un ping a google per comprovar la conexió ethernet.

    sudo nano /etc/hostname


    sudo nano /etc/hosts

    Desprès, fem un sudo apt update per actualitzar els canvis.

    sudo apt install sladp ldap-utils

    Configurem amb una contrasenya per l'administrador:

    Confirmem la contrasenya:

    Per comprovar els canvis la comanda: 

    sudo slapcat

    Hem d'executar aquesta comanda en root, sinò ens enviarà un error d'insuficients permisos. 

    dpkg-reconfigure slapd

    1.2 Crear una UO

    Amb crearem el nostre arxiu de l'unitat organitzativa dels usuaris 

    nano uo.ldif


    ldapadd -c -x -D "cn=admin,dc=sekiro,dc=cat" -W -f uo.ldif

    1.3 Crear Usuaris

    nano usu.ldif

    Utilitzem la mateixa comanda amb el nou arxiu .ldif 

    ldapadd -c -x -D "cn=admin,dc=sekiro,dc=cat" -W -f usu.ldif

    1.4 Crear Grup

    nano grup.ldif

    1.4.1 Comprovació

    Finalment, fem la comprovacio amb sudo slapcat.

    2. UNIR EQUIPS DOMINI

    Amb la comanda sudo su abrim la nostra terminal amb l'usuari root.

    apt install nscd


    apt install libnss-ldap

    Sempre que quan lo client no es pugui validar al domini a part de comprovar fitxers, també farem sempre un reconfigure. 

    dpkg-reconfigure ldap-auth-config

    Reconfigurem els paràmetre que vulguem de cada pas i al final fem clic a "Ok" per continuar.

    nano /etc/nsswitch.conf
