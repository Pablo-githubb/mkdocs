    GESTIÓ I IDENTIFICACIÓ DE PROCESSOS

    Alhora d'identificar procesos podem utilitzar la comanda pstree. Però, per no tindre incompatibilitats utilitzarem la comanda sudo su per estar en superadmin.

    Ara si utilitzem pstree -h "usuari" podrem veure els procesos executats per l'usuari indicat (en el meu cas, pmc)

    A més a més, utilitzant la comanda pstree -h -p "usuari" podem a més veure els procesos de l'usuari amb el ID de cada proces.

    NF4-USUARIS-LINUX (1-9. 13-15. 26-29)

    Crear nous usuaris

    adduser gina addgroup cat /etc/passwd | grep gina cat /etc/group | grep gina cat /etc/gshadow | grep gina

    Afegir usuaris al sudo

    adduser gina sudo deluser gina sudo //treure a gina del group sudo deluser gina //borrar usuari passwd gina //canviar contrasenya del usuari

    gpasswd -a user group

    Borrar usuaris

    deluser gina userdel -r gina //borrar l'usuari i el seu /home deluser gina group chown julia:julia julia //canviar de propietari la carpeta creada a un usuari en concret

    Modificació dels usuaris i grups

    usermod -p contrasenya julia usermod -s interprete usermode -L user //bloqueja usermod -U gina usermod --help sudo mv /home/julia julia2 Moure la carpeta física usermod -d /home/julia2 julia Modificar en el /passwd groupmode -n julia2 julia

    usermod -a -G group user gpasswd -A user group //afegir un user a un group com admin usermod -g group user // afegit al grup principal del usuari groupmod -n castanyada halloween // canvia el nom del grup
