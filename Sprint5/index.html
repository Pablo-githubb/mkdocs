    Sprint 5 (Projecte 2 Sprint1)

    1. Instal·lació Windows 10/Windows Server

    1.1 Instal·lació bàsica Windows 10 Enterprise

    Al iniciar la nostra màquina, ens arribarà automàticament a aquesta finestra per sel·leccionar l'idioma, el format de moneda i la distribució del teclat.

    Següent, fem clic al botó central d'instal·lar.

    Ara, sel·leccionem la segona opció Personalizada solo instalar Windows (avanzado).

    Dintre d'aqui, creem una partició sense formatar amb la meitat de disc de la nostra partició principal (en el meu cas era de 50gb. Llavors, utilitzo 25gb).

    Ja amb el procès d'instal·lació finalitzat, solament falta configurar l'usuari principal, li fiquem un nom d'usuari:

    Li fiquem una constrasenya.

    Finalment, unes preguntes de seguretat per l'usuari.

    1.2 Instal·lació bàsica Windows Server 2022

    Al començar l'instal·lació del Windows Server, ens obrirà la mateixa finestra sobre l'idioma, distribució de teclat, etc.

    A continuació, sel·leccionem el la versió Standard Evaluation amb experiencia de escritorio.

    Com amb el Windows 10, sel·leccionem també Personalizada. Però, no separem la partició principal.

    Al acabar l'instal·lació, configurem l'usuari Administrador amb una constrasenya i ja tindrem el Windows Server 2022 configura't.

    2. Configuració de la xarxa

    2.1 Configuració de la xarxa Windows Server

    Obrim la configuració de xarxa:

    Prenem cambiar opciones de adaptador.

    Entrem dintre de la configuració d'adaptador i en les seves propietats.

    Dintre de Protocolo de Internet version 4 (TCP/IPv4).

    Configurem la ip estàtica amb la xarxa NAT 10.0.2.15/24 i 10.0.2.1. El DNS serà el propi servidor 10.0.2.15.

    2.2 Configuració de la xarxa Windows Enterprise

    Finalment, en l'apartat de DNS en el Windows 10, assignem la IP del Windows Server com DNS principal. L'altra podem ficar per exemple Google (8.8.8.8).
